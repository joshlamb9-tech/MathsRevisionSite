Key Facts

  • Mean (from a list): Add all values and divide by how many there are. $$\text{Mean} = \frac{\text{sum of values}}{\text{number of values}}$$
  • Mean from a frequency table: Multiply each value by its frequency, add these products, then divide by the total frequency. $$\text{Mean} = \frac{\sum fx}{\sum f}$$ where $f$ is the frequency and $x$ is the value.
  • Median: The middle value when all data is in order. If there are $n$ values, the median is in position $\dfrac{n+1}{2}$.
  • Mode: The value that occurs most often. There can be more than one mode. In a frequency table, the mode is the value with the highest frequency.
  • Range: $\text{Range} = \text{largest value} - \text{smallest value}$. Measures spread, not average.

Worked Example 2: Median from an Ordered List

Find the median of: 3, 7, 2, 9, 4, 1, 8, 5, 6

  1. Write the values in ascending order: 1, 2, 3, 4, 5, 6, 7, 8, 9
  2. There are $n = 9$ values. The median is in position $\dfrac{9+1}{2} = 5$.
  3. Count to the 5th value in the ordered list: 1, 2, 3, 4, 5

Answer: Median = 5
