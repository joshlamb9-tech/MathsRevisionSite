Key Facts

  • Equation of a straight line: $y = mx + c$, where $m$ is the gradient and $c$ is the $y$-intercept (where the line crosses the $y$-axis).
  • Gradient: How steep the line is. Positive gradient = slopes up left to right. Negative gradient = slopes down left to right. $$m = \frac{\text{rise}}{\text{run}} = \frac{y_2 - y_1}{x_2 - x_1}$$
  • y-intercept: The value of $y$ when $x = 0$. On the graph this is where the line crosses the vertical axis.
  • Plotting a straight line: Use the $y$-intercept to mark a starting point on the $y$-axis. Then use the gradient to find two more points — rise over run. Join the points with a ruler.
  • Parallel lines: Two lines are parallel if and only if they have the same gradient $m$. Example: $y = 3x + 1$ and $y = 3x - 4$ are parallel.

Worked Example 1: Plotting a Straight Line

Plot the line $y = 2x - 1$. State its gradient and $y$-intercept.

  1. Identify gradient and $y$-intercept from the equation $y = 2x - 1$: gradient $m = 2$, $y$-intercept $c = -1$.
  2. Mark the $y$-intercept at $(0, -1)$.
  3. Use the gradient: from $(0, -1)$, go 1 right and 2 up to reach $(1, 1)$. Repeat to get $(2, 3)$.
  4. Build a table of values to confirm:
    When $x = 0$: $y = 2(0) - 1 = -1$, giving point $(0, -1)$
    When $x = 1$: $y = 2(1) - 1 = 1$, giving point $(1, 1)$
    When $x = 2$: $y = 2(2) - 1 = 3$, giving point $(2, 3)$
  5. Plot the three points and draw a straight line through them.

Answer: gradient = 2 (rises 2 for every 1 across), $y$-intercept = $-1$.

Worked Example 2: Finding the Equation of a Line

Find the equation of the straight line that passes through $(0, 3)$ and $(4, 11)$.

  1. Calculate the gradient using the two points: $$m = \frac{11 - 3}{4 - 0} = \frac{8}{4} = 2$$
  2. The first point $(0, 3)$ is on the $y$-axis, so the $y$-intercept is $c = 3$.
  3. Write the equation: $y = mx + c = 2x + 3$
  4. Check: substitute $x = 4$: $y = 2(4) + 3 = 11$ ✓

Answer: $y = 2x + 3$
