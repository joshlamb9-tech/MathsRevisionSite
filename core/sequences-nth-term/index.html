Key Facts

  • Arithmetic sequence: A sequence where the difference between consecutive terms is constant. This constant difference is called $d$ (the common difference). Example: 3, 7, 11, 15, ... has $d = 4$.
  • Finding the nth term: For an arithmetic sequence with first term $a$ and common difference $d$: $$U_n = dn + (a - d)$$ A quicker way to remember: $U_n = d \cdot n + c$, where $c$ is the "zeroth term" (the value before the sequence starts, found by subtracting $d$ from the first term).
  • Using the nth term: Substitute any value of $n$ to find that term. For example, $U_{10}$ means the 10th term â€” substitute $n = 10$.
  • Is a value in the sequence? Set $U_n$ equal to the value, solve for $n$. If $n$ is a positive integer, the value is in the sequence.
  • Square and triangle numbers: These are non-arithmetic sequences worth knowing. Square numbers: 1, 4, 9, 16, 25, ... ($U_n = n^2$). Triangle numbers: 1, 3, 6, 10, 15, ... ($U_n = \frac{1}{2}n(n+1)$).

Worked Example 1: Finding the Nth Term

Find the nth term formula for the sequence 5, 8, 11, 14, ...

  1. Find the common difference: $d = 8 - 5 = 3$
  2. Find the zeroth term (subtract $d$ from the first term): $a - d = 5 - 3 = 2$
  3. Write the formula: $$U_n = 3n + 2$$
  4. Check by substituting $n = 1$: $U_1 = 3(1) + 2 = 5$ ✓ and $n = 2$: $U_2 = 3(2) + 2 = 8$ ✓

Answer: $U_n = 3n + 2$

Worked Example 2: Is a Value in the Sequence?

The nth term of a sequence is $U_n = 4n - 3$. Is 100 a term in this sequence?

  1. Set the formula equal to 100 and solve for $n$: $$4n - 3 = 100$$
  2. Add 3 to both sides: $4n = 103$
  3. Divide by 4: $$n = \frac{103}{4} = 25.75$$
  4. Since $n = 25.75$ is not a positive integer (whole number), 100 is not a term in this sequence.

Answer: 100 is not a term in the sequence (it falls between the 25th and 26th terms).
