Key Facts

  • Collecting like terms: Only add or subtract terms with the same variable and power. Example: $3x + 5x = 8x$, but $3x + 5y$ cannot be simplified.
  • Expanding a bracket: Multiply the term outside the bracket by every term inside. Example: $3(2x - 4) = 6x - 12$.
  • Solving linear equations: Do the same operation to both sides to keep the equation balanced. Aim to get the variable alone on one side.
  • Substitution: Replace the variable with a given number and evaluate. Example: if $x = 3$, then $2x^2 - 1 = 2(9) - 1 = 17$.
  • Forming expressions: Translate word problems into algebra. "Five more than three times a number $n$" becomes $3n + 5$.

Worked Example 1: Solving a Linear Equation

Solve $3x + 7 = 22$.

  1. Subtract 7 from both sides: $$3x + 7 - 7 = 22 - 7$$ $$3x = 15$$
  2. Divide both sides by 3: $$x = \frac{15}{3} = 5$$
  3. Check by substituting back: $3(5) + 7 = 15 + 7 = 22$ ✓

Answer: $x = 5$

Worked Example 2: Expanding and Simplifying

Expand and simplify $3(2x - 1) + 4(x + 3)$.

  1. Expand the first bracket: $3(2x - 1) = 6x - 3$
  2. Expand the second bracket: $4(x + 3) = 4x + 12$
  3. Collect all the terms: $$6x - 3 + 4x + 12$$
  4. Collect like terms (x terms together, number terms together): $$6x + 4x = 10x \qquad -3 + 12 = 9$$

Answer: $10x + 9$
