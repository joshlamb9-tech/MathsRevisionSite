Key Facts

  • Coordinate pairs: Written as $(x, y)$. The $x$-value (horizontal) always comes first. Example: the point $(3, -2)$ is 3 right and 2 down from the origin.
  • Four quadrants: The axes divide the plane into four quadrants. Top-right: $(+, +)$; top-left: $(-, +)$; bottom-left: $(-, -)$; bottom-right: $(+, -)$.
  • Origin: The point $(0, 0)$ where the $x$-axis and $y$-axis cross.
  • Midpoint formula: The midpoint of the line segment from $A(x_1, y_1)$ to $B(x_2, y_2)$ is: $$M = \left(\frac{x_1 + x_2}{2},\ \frac{y_1 + y_2}{2}\right)$$
  • Plotting a point: Start at the origin. Move horizontally by the $x$-value, then vertically by the $y$-value.

The Four Quadrants

Worked Example 1: Identifying Quadrants

Plot the points $A(2, 3)$, $B(-1, 4)$, $C(-2, -3)$, $D(4, -1)$ and state which quadrant each point is in.

  1. $A(2, 3)$: $x$ is positive, $y$ is positive → Quadrant 1 (top-right)
  2. $B(-1, 4)$: $x$ is negative, $y$ is positive → Quadrant 2 (top-left)
  3. $C(-2, -3)$: $x$ is negative, $y$ is negative → Quadrant 3 (bottom-left)
  4. $D(4, -1)$: $x$ is positive, $y$ is negative → Quadrant 4 (bottom-right)

Answer: A — Q1, B — Q2, C — Q3, D — Q4.

Worked Example 2: Finding a Midpoint

Find the midpoint of the line segment from $A(1, 5)$ to $B(7, -1)$.

  1. Apply the midpoint formula with $x_1 = 1$, $y_1 = 5$, $x_2 = 7$, $y_2 = -1$: $$M = \left(\frac{1 + 7}{2},\ \frac{5 + (-1)}{2}\right)$$
  2. Simplify each coordinate: $$M = \left(\frac{8}{2},\ \frac{4}{2}\right) = (4,\ 2)$$

Answer: Midpoint $M = (4, 2)$
