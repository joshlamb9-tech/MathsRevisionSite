Long Multiplication

Key Facts

  • Column method: Write the larger number on top. Multiply by the units digit first, then by the tens digit (adding a zero placeholder on the second row), then add the two rows.
  • Zero placeholder: When multiplying by the tens digit, put a 0 in the units column of the second row before multiplying.
  • Grid method (alternative): Partition each number into hundreds, tens, and units. Multiply each combination and add all the partial products.
  • Always estimate first to check your answer is reasonable. $47 \times 36 \approx 50 \times 40 = 2000$

Worked Example 1: 2-Digit × 2-Digit (Column Method)

Calculate $\mathbf{47 \times 36}$ using the column method.

  1. Estimate first: $50 \times 36 \approx 1800$ (so the answer should be near this)
  2. Multiply by the units digit (6):
    • $6 \times 7 = 42$ — write 2, carry 4
    • $6 \times 4 = 24$, plus carried 4 $= 28$ — write 28
    • First row: $\mathbf{282}$
  3. Multiply by the tens digit (3), add a 0 placeholder:
    • $3 \times 7 = 21$ — write 1, carry 2
    • $3 \times 4 = 12$, plus carried 2 $= 14$ — write 14
    • Second row: $\mathbf{1410}$ (the 0 is the placeholder)
  4. Add the two rows: $$282 + 1410 = 1692$$

Answer: $47 \times 36 = 1692$

Worked Example 2: 3-Digit × 2-Digit (Column Method)

Calculate $\mathbf{324 \times 53}$.

  1. Estimate: $300 \times 50 = 15\,000$
  2. Multiply by 3 (units):
    • $3 \times 4 = 12$ — write 2, carry 1
    • $3 \times 2 = 6$, plus 1 $= 7$
    • $3 \times 3 = 9$
    • First row: $\mathbf{972}$
  3. Multiply by 5 (tens), add a 0 placeholder:
    • $5 \times 4 = 20$ — write 0, carry 2
    • $5 \times 2 = 10$, plus 2 $= 12$ — write 2, carry 1
    • $5 \times 3 = 15$, plus 1 $= 16$
    • Second row: $\mathbf{16200}$
  4. Add: $$972 + 16200 = 17172$$

Answer: $324 \times 53 = 17\,172$

Check: estimate was 15,000 — answer is 17,172, which is close. ✓
