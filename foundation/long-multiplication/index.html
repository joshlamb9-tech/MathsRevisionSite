Key Facts

  • Column method: Write the larger number on top. Multiply by the units digit first, then by the tens digit (adding a zero placeholder on the second row), then add the two rows.
  • Zero placeholder: When multiplying by the tens digit, put a 0 in the units column of the second row before multiplying.
  • Grid method (alternative): Partition each number into hundreds, tens, and units. Multiply each combination and add all the partial products.
  • Always estimate first to check your answer is reasonable. $47 \times 36 \approx 50 \times 40 = 2000$

Worked Example 1: 2-Digit × 2-Digit (Column Method)

Calculate $\mathbf{47 \times 36}$ using the column method.

Estimate first: $50 \times 40 \approx 2000$ — so the answer should be near this.

Write one number on top of the other, digits aligned:
× 4 7
× 3 6
row 1 (×6): 2 8 2 (carry 4)
row 2 (×30): 1 4 1 0 (0 placeholder)
1 6 9 2
  1. Multiply by 6 (units digit): $6 \times 7 = 42$ — write 2, carry 4. Then $6 \times 4 = 24 + 4 = 28$. First row: 282.
  2. Multiply by 3 (tens digit): Write a 0 placeholder in the units column first. Then $3 \times 7 = 21$ — write 1, carry 2. Then $3 \times 4 = 12 + 2 = 14$. Second row: 1410.
  3. Add the rows: $282 + 1410 = 1692$.

Answer: $47 \times 36 = 1692$   Check: estimate was ~2000. ✓

Worked Example 2: 3-Digit × 2-Digit (Column Method)

Calculate $\mathbf{324 \times 53}$.

Estimate first: $300 \times 50 = 15\,000$.

3 2 4
× 5 3
row 1 (×3): 9 7 2
row 2 (×50): 1 6 2 0 0
1 7 1 7 2
  1. Multiply by 3 (units): $3 \times 4 = 12$ — write 2, carry 1. $3 \times 2 = 6 + 1 = 7$. $3 \times 3 = 9$. Row 1: 972.
  2. Multiply by 5 (tens): Write a 0 placeholder first. $5 \times 4 = 20$ — write 0, carry 2. $5 \times 2 = 10 + 2 = 12$ — write 2, carry 1. $5 \times 3 = 15 + 1 = 16$. Row 2: 16200.
  3. Add: $972 + 16200 = 17\,172$.

Answer: $324 \times 53 = 17\,172$   Check: estimate was 15,000 — close. ✓

Quick Check

Estimate $63 \times 49$, then calculate the exact answer.

Estimate: $60 \times 50 = 3{,}000$
Exact: Row 1 ($\times 9$): $567$. Row 2 ($\times 60$): $3780$.
$567 + 3780 = \mathbf{3{,}087}$

Calculate $84 \times 36$.

Row 1 ($\times 6$): $504$.
Row 2 ($\times 30$): $2520$.
$504 + 2520 = \mathbf{3{,}024}$

A school orders 42 boxes each containing 125 pencils. How many pencils in total?

Row 1 ($\times 2$): $250$.
Row 2 ($\times 40$): $5000$.
$250 + 5000 = \mathbf{5{,}250}$ pencils
