Functions and Graph Transformations

Key Facts

  • Function notation: $f(x)$ means "a function of $x$". If $f(x) = 2x + 3$, then $f(4) = 2(4) + 3 = 11$.
  • $f(x) + a$ — translates the graph up by $a$ units (or down if $a$ is negative).
  • $f(x + a)$ — translates the graph left by $a$ units (note: the opposite direction to what you might expect).
  • $-f(x)$ — reflects the graph in the $x$-axis (flips it vertically).
  • $f(-x)$ — reflects the graph in the $y$-axis (flips it horizontally).

Worked Example 1: Function Evaluation

$f(x) = x^2 - 3x + 1$. Find $f(5)$ and $f(-2)$.

  1. $f(5)$: Replace $x$ with 5: $$f(5) = 5^2 - 3(5) + 1 = 25 - 15 + 1 = 11$$
  2. $f(-2)$: Replace $x$ with $-2$: $$f(-2) = (-2)^2 - 3(-2) + 1 = 4 + 6 + 1 = 11$$

Answers: $f(5) = 11$ and $f(-2) = 11$

Worked Example 2: Graph Transformations

The graph of $y = f(x)$ passes through the point $(3, 7)$. State the corresponding point on each transformed graph.

  1. $y = f(x) + 4$: Shift up 4 → $(3, 11)$
  2. $y = f(x + 2)$: Shift left 2 → $(1, 7)$
  3. $y = -f(x)$: Reflect in $x$-axis → $(3, -7)$
  4. $y = f(-x)$: Reflect in $y$-axis → $(-3, 7)$

Answers: $(3, 11)$, $(1, 7)$, $(3, -7)$, $(-3, 7)$
