Key Facts

  • Standard form: A quadratic equation has the form $ax^2 + bx + c = 0$ where $a \neq 0$.
  • Solving by factorising: If $ax^2 + bx + c$ factorises, set each bracket equal to zero. For example, if $(x+2)(x-3) = 0$, then $x = -2$ or $x = 3$.
  • The quadratic formula: When the expression does not factorise neatly: $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$ The $\pm$ means there are usually two solutions.
  • Always rearrange to $= 0$ first before factorising or using the formula.

Worked Example 1: Solving by Factorising

Solve $x^2 - 5x + 6 = 0$.

  1. Find two numbers that multiply to 6 and add to $-5$: $-2$ and $-3$ (since $(-2) \times (-3) = 6$ and $(-2) + (-3) = -5$).
  2. Factorise: $$(x - 2)(x - 3) = 0$$
  3. Set each bracket to zero: $$x - 2 = 0 \implies x = 2 \qquad \text{or} \qquad x - 3 = 0 \implies x = 3$$

Answer: $x = 2$ or $x = 3$

Worked Example 2: Using the Quadratic Formula

Solve $2x^2 + 3x - 5 = 0$. Give answers to 2 decimal places.

  1. Identify $a$, $b$, $c$: $a = 2$, $b = 3$, $c = -5$.
  2. Substitute into the formula: $$x = \frac{-3 \pm \sqrt{3^2 - 4(2)(-5)}}{2(2)} = \frac{-3 \pm \sqrt{9 + 40}}{4} = \frac{-3 \pm \sqrt{49}}{4} = \frac{-3 \pm 7}{4}$$
  3. Two solutions: $$x = \frac{-3 + 7}{4} = \frac{4}{4} = 1 \qquad \text{or} \qquad x = \frac{-3 - 7}{4} = \frac{-10}{4} = -2.5$$

Answer: $x = 1$ or $x = -2.5$
