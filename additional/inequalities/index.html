Key Facts

  • Inequality symbols: $<$ means "less than", $>$ means "greater than", $\leq$ means "less than or equal to", $\geq$ means "greater than or equal to".
  • Solving inequalities: Treat them exactly like equations — add, subtract, multiply, divide both sides. But if you multiply or divide by a negative number, flip the inequality sign.
  • Number line representation: Use an open circle ○ for $<$ or $>$ (not included) and a filled circle ● for $\leq$ or $\geq$ (included).
  • Double inequalities: $a < x < b$ means $x$ is between $a$ and $b$. Solve by doing the same operation to all three parts.

Worked Example 1: Solving a Linear Inequality

Solve $3x + 5 < 20$.

  1. Subtract 5 from both sides: $$3x < 15$$
  2. Divide both sides by 3: $$x < 5$$

Answer: $x < 5$

Worked Example 2: Double Inequality

Solve $-3 \leq 2x + 1 < 9$. List the integer values of $x$.

  1. Subtract 1 from all three parts: $$-4 \leq 2x < 8$$
  2. Divide all three parts by 2: $$-2 \leq x < 4$$
  3. Integer values: $x$ can be $-2, -1, 0, 1, 2, 3$ (note: $x = 4$ is not included because of the strict $<$).

Answer: $-2 \leq x < 4$; integers: $-2, -1, 0, 1, 2, 3$
