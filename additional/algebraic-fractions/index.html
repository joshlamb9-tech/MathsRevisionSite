Algebraic Fractions

Key Facts

  • Simplifying: Factorise the numerator and denominator, then cancel common factors: $$\frac{x^2 + 3x}{x} = \frac{x(x + 3)}{x} = x + 3$$
  • Multiplying: Multiply the numerators together and the denominators together, then simplify: $$\frac{a}{b} \times \frac{c}{d} = \frac{ac}{bd}$$
  • Dividing: Flip the second fraction and multiply: $$\frac{a}{b} \div \frac{c}{d} = \frac{a}{b} \times \frac{d}{c} = \frac{ad}{bc}$$
  • Adding and subtracting: Find a common denominator, just as with numerical fractions: $$\frac{a}{b} + \frac{c}{d} = \frac{ad + bc}{bd}$$

Worked Example 1: Simplifying

Simplify $\dfrac{2x^2 + 6x}{4x}$.

  1. Factorise the numerator: $$2x^2 + 6x = 2x(x + 3)$$
  2. Write as a fraction and cancel: $$\frac{2x(x + 3)}{4x} = \frac{x + 3}{2}$$

Answer: $\dfrac{x + 3}{2}$

Worked Example 2: Adding Algebraic Fractions

Simplify $\dfrac{3}{x} + \dfrac{2}{x+1}$.

  1. Common denominator: $x(x+1)$
  2. Rewrite each fraction: $$\frac{3(x+1)}{x(x+1)} + \frac{2x}{x(x+1)}$$
  3. Combine: $$\frac{3(x+1) + 2x}{x(x+1)} = \frac{3x + 3 + 2x}{x(x+1)} = \frac{5x + 3}{x(x+1)}$$

Answer: $\dfrac{5x + 3}{x(x+1)}$
