Direct and Inverse Proportion

Key Facts

  • Direct proportion: As one quantity increases, the other increases at the same rate. Written as $y \propto x$, which means $y = kx$ for some constant $k$.
  • Inverse proportion: As one quantity increases, the other decreases. Written as $y \propto \dfrac{1}{x}$, which means $y = \dfrac{k}{x}$.
  • Finding $k$: Substitute a known pair of values to find the constant $k$, then use it to find unknowns.
  • Recognising the type: If doubling $x$ doubles $y$ → direct. If doubling $x$ halves $y$ → inverse.

Worked Example 1: Direct Proportion

$y$ is directly proportional to $x$. When $x = 4$, $y = 20$. Find $y$ when $x = 7$.

  1. Write the formula: $y = kx$
  2. Find $k$: $20 = k \times 4$, so $k = 5$
  3. Use the formula: $y = 5 \times 7 = 35$

Answer: $y = 35$

Worked Example 2: Inverse Proportion

$y$ is inversely proportional to $x$. When $x = 3$, $y = 12$. Find $y$ when $x = 9$.

  1. Write the formula: $y = \dfrac{k}{x}$
  2. Find $k$: $12 = \dfrac{k}{3}$, so $k = 36$
  3. Use the formula: $y = \dfrac{36}{9} = 4$
  4. Check: $x$ tripled ($3 \to 9$), so $y$ should be divided by 3 ($12 \div 3 = 4$) ✓

Answer: $y = 4$
