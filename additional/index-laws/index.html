Key Facts

  • Multiplication law — when multiplying powers with the same base, add the indices: $$a^m \times a^n = a^{m+n}$$ The bases must be the same. You cannot combine $2^3 \times 3^4$ using this law.
  • Division law — when dividing powers with the same base, subtract the indices: $$a^m \div a^n = a^{m-n}$$
  • Power of a power — when raising a power to another power, multiply the indices: $$(a^m)^n = a^{mn}$$
  • Zero index — any non-zero number raised to the power zero equals 1: $a^0 = 1$ for any $a \neq 0$. For example, $5^0 = 1$ and $100^0 = 1$.
  • Negative index — a negative index means "one over" the positive power: $$a^{-n} = \dfrac{1}{a^n}$$ For example, $2^{-3} = \dfrac{1}{2^3} = \dfrac{1}{8}$.

Worked Example 1: Multiplication and Division Laws

Simplify $5^3 \times 5^4$ and $\dfrac{2^7}{2^3}$.

  1. Multiplication: Both terms have base 5, so apply the multiplication law — add the indices: $$5^3 \times 5^4 = 5^{3+4} = 5^7$$
  2. Division: Both terms have base 2, so apply the division law — subtract the indices: $$\frac{2^7}{2^3} = 2^{7-3} = 2^4 = 16$$

Answers: $5^7$ and $16$

Worked Example 2: Power of a Power, Zero Index, and Negative Index

Simplify $(3^2)^4$. Evaluate $7^0$. Write $3^{-2}$ as a fraction.

  1. Power of a power: Multiply the indices: $$(3^2)^4 = 3^{2 \times 4} = 3^8$$
  2. Zero index: Any non-zero base raised to the power zero equals 1: $$7^0 = 1$$
  3. Negative index: Flip to a fraction using the positive power: $$3^{-2} = \dfrac{1}{3^2} = \dfrac{1}{9}$$

Answers: $3^8$, $1$, and $\dfrac{1}{9}$
