Expanding and Factorising

Key Facts

  • Expanding double brackets (FOIL): Multiply every term in the first bracket by every term in the second bracket using the order First, Outer, Inner, Last: $$(x + a)(x + b) = x^2 + (a+b)x + ab$$ Collect like terms after expanding.
  • Factorising $x^2 + bx + c$: Find two numbers that multiply to $c$ and add to $b$. If those numbers are $p$ and $q$, then: $$x^2 + bx + c = (x + p)(x + q)$$ Always check by expanding back out.
  • Difference of two squares: A special pattern — the factorisation of $a^2 - b^2$ is: $$a^2 - b^2 = (a + b)(a - b)$$ Example: $x^2 - 9 = (x + 3)(x - 3)$

Worked Example 1: Expanding $(x + 3)(x - 2)$

Use FOIL to expand the double brackets, then collect like terms.

  1. First: Multiply the first term of each bracket: $x \times x = x^2$
  2. Outer: Multiply the outer terms: $x \times (-2) = -2x$
  3. Inner: Multiply the inner terms: $3 \times x = 3x$
  4. Last: Multiply the last term of each bracket: $3 \times (-2) = -6$
  5. Collect like terms — combine the $-2x$ and $+3x$: $$x^2 - 2x + 3x - 6 = x^2 + x - 6$$

Answer: $x^2 + x - 6$

Worked Example 2: Factorising $x^2 + 5x + 6$

Find two numbers that multiply to $6$ and add to $5$, then write the factorised form and verify by expanding.

  1. List factor pairs of 6: $(1, 6)$ and $(2, 3)$. Check which pair adds to 5: $1 + 6 = 7$ (no)   $2 + 3 = 5$ (yes!). The two numbers are $2$ and $3$: $2 \times 3 = 6$ and $2 + 3 = 5$.
  2. Write the factorised form: $$x^2 + 5x + 6 = (x + 2)(x + 3)$$
  3. Check by expanding (FOIL): $$(x + 2)(x + 3) = x^2 + 3x + 2x + 6 = x^2 + 5x + 6 \checkmark$$ The expansion matches the original expression — the factorisation is correct.

Answer: $(x + 2)(x + 3)$
